# Diabetes Patient Clustering
Unsupervised machine learning project using K-Means, 
Hierarchical, and DBSCAN clustering on the Pima Indians 
Diabetes Dataset (768 patients, 8 features).

## Key Results
- Optimal k=3 clusters (Low/Medium/High Risk)
- Silhouette Score: 0.2479
- Calinski-Harabasz: 242.7
